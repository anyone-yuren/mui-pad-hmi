## 使用

### clone

```bash
$ git clone https://github.com/anyone-yuren/mui-pad-hmi.git
```

### yarn

```bash
$ yarn install
$ yarn dev
```

### or pnpm

```
$ pnpm i
$ pnpm dev
```
